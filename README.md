# Viro Xyz 🌌

**Viro Xyz** is a terminal-style personal website hosting projects, downloads, and experiments. With black-and-white aesthetics, blinking cursors, and retro CRT effects, it’s a digital hub for learning, exploring, and experimenting with web design in a nostalgic, terminal-inspired environment.

---

## 🚀 Features

- **Terminal-inspired UI:** Monospace fonts, blinking cursors, and command-like buttons.  
- **Homepage:** Introductory terminal hub with navigation commands and welcome message.  
- **Downloads:** Access hosted files for learning or experimentation.  
- **Bio:** Personal info and digital projects.  
- **Other:** Sandbox for miscellaneous experiments and creative content.  
- **Background Effects:** Broken screen overlay, subtle rain, and retro CRT flicker (on homepage).  
- **Interactive Buttons:** Command-style buttons with blinking cursor effect.

---

## 📄 Extra Website Info

Viro Xyz is designed not just as a portfolio, but as a **learning environment for web enthusiasts**. It demonstrates:  
- Terminal-style design and animations  
- CSS + JS visual effects like rain, flicker, and cracked screen overlays  
- Interactive navigation that mimics real terminal commands  
- Responsive and minimalistic layout optimized for both desktop and mobile  

Whether you’re exploring HTML/CSS/JS or just enjoy retro terminal aesthetics, this website is a fun and educational experience.
